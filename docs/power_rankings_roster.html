Methodology & Config
Each unit score is a position-weighted blend of starter-quality ratings for that side of the ball, boosted by premium dev traits at high OVR bands and then normalized to a 0–100 scale. By default we use a z-score style normalization (mean around 50, spread by league-wide standard deviation), but the CLI can also be configured to use a simple min–max 0–100 stretch.

Units & position weighting

  • Off Pass (QB/WR/TE/OL): QB 45%, WR 30%, OL 15%, TE 10%
  • Off Run (RB/FB/TE/OL): OL 40%, RB 35%, TE 15%, FB 10%
  • Pass Coverage (CB/S/LB): CB 50%, S 30%, LB 20%
  • Pass Rush (DE/DT/LB): DE 40%, DT 30%, LB 30%
  • Run Defense (DE/DT/LB/S): DT 35%, LB 30%, DE 25%, S 10%

Dev traits impact

  • X-Factor: +10% above 85 OVR, +5% above 80 OVR
  • Superstar: +7% above 85 OVR, +3% above 80 OVR
  • Star: baseline impact only (no extra multiplier by default).
  • Normal: baseline impact only (no extra multiplier by default).

Normalization & overall score

Overall roster power combines normalized unit scores using: Off Pass 30% + Off Run 20% + Pass Coverage 30% + Pass Rush 20%. Run Defense is shown as its own unit but is not included in the default composite score, but can be folded in by changing the overall unit weights.

Normalization method (per run): z-score (mean ~50, spread by league standard deviation) or optional min–max (min team = 0, max team = 100).
